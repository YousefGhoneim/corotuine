# 🛍️ Android Product App

An Android Application that helps the user view all products from an API, add desired products to favorites, and remove them anytime.

## 🧠 Tech Stack

- Kotlin
- MVVM
- Room
- Retrofit
- Gson
- Coroutine
- Glide
- Navigation Component
- minSdk 24, targetSdk 34
